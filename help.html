NHentaiAnalytics is a Chrome/Firefox extension to help you find the best doujinshi based on your favorites.
If you have a problem, try reloading the doujinshi list by going in settings then press "Update".
NHentaiAnalytics is a open-source project under the Apache License 2.0.
The source code is freely available on GitHub: https://github.com/Xwilarg/NHentaiAnalytics.
