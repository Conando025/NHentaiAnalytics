Explanations:
The extension take all the tags from your favorites doujinshi and order them in descending order.
When doing the requests to get each tag, we need to wait between each requests to each doujinshi (so we don't get a 50X error)
Then it takes X tags from the top of the list, these tags are considered as your favorite tags.
From these X tags, it then take Y random tags and do a search with them.
After that it just take a random doujinshi and show it to you.
